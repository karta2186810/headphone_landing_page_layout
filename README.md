# Headphone RWD Landing Page Layout
### 介紹
耳機 Landing Page 切版練習作品，
用以學習 CSS Variables 的使用、CSS BEM 設計模式、 Grid 排版的使用，以及 Mobile First RWD 網頁設計。

### 使用語言
* HTML
* CSS
* JavaScript

### 動畫效果
* 原生JavaScript
* Scroll Reveal

### 構建工具
* Webpack
* JS兼容性處理: Babel
* CSS前綴處理: PostCss
